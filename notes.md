# Bash Notes
- ~  - HOME directory
- /  - absolute path that starts at root
- .  - relative path that starts at current directory
	- .. - previous directory

- pwd - Print working directory
- cd - change directory
- mkdir - make directory
- rm - remove
	- -r - recursively (delete everything inside)
	- -f - force
- rmdir - remove a directory
- touch - create a file
- ls - list contents of a directory
	- -a - list hidden files
	- -l - include information about the contents
- nano - open nano editor
	- ^ is ctrl
- vim - open vim editor

## Path Variable
- Every available path that can be accessed

## Git
- git init - create repository
- git add - stage changes
	- rm cached - remove staged file
- git commit
	- -m "" - to commit message
